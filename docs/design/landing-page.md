# Landing Page

## Zweck

Die Landing Page erklärt Sportify in wenigen Sekunden: **Was**, **für wen**, **warum**.

## Struktur (siehe `src/pages/Landing.tsx`)

- Hero mit klarer Value Proposition
- Feature‑Sektionen (Metriken, Nutzen, Social Proof)
- CTA‑Blöcke, klare Navigation

## Tonalität

- sportlich, motivierend, vertrauenswürdig

## Begründung

Die Landing Page ist der erste Kontaktpunkt. Sie muss Vertrauen schaffen und die App als leistungsstarkes, aber einfaches Tool positionieren.
