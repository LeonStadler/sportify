# Layout & Grid

## Prinzipien

- **Lesefluss vor Dichte**: Inhalte müssen schnell scanbar sein.
- **Konsistente Vertikale**: Rhythmus über einheitliche Abstände.
- **Responsive zuerst**: Mobile‑Nutzung ist Kernziel.

## Breakpoints

- xs 480px
- sm 640px
- md 768px
- lg 1024px
- xl 1280px
- 2xl 1536px

## Container

- Centered
- Padding 2rem
- Max‑Width 1400px (2xl)

Begründung: Breite Layouts geben Charts Raum, ohne Textzeilen zu lang zu machen.

## Grid

- Desktop: 12‑Spalten
- Tablet: 8‑Spalten
- Mobile: 4–6‑Spalten

## Spacing

- 4px‑Basis
- Wichtigste Stufen: 4/8/12/16/24/32/48/64
- Safe‑Area‑Padding: `safe-top/bottom/left/right`

## Radius

- `--radius`: 0.5rem

Begründung: Mildes Rounded‑Design wirkt modern und zugänglich.
